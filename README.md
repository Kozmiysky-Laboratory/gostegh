# ГОСТегх (GOSTegh)

Автоматическое форматирование документов по ГОСТ 7.32-2017.

## Функционал

Сервис реализует два базовых пункта форматирования по ГОСТ 7.32-2017:

1. **Поля страницы:**
   - Левое: 30 мм
   - Правое: 15 мм
   - Верхнее: 20 мм
   - Нижнее: 20 мм

2. **Глобальный стиль текста:**
   - Шрифт: Times New Roman
   - Размер: 14 pt
   - Межстрочный интервал: 1.5
   - Абзацный отступ (красная строка): 1.25 см
   - Выравнивание: по ширине

## Безопасность

Файлы обрабатываются **строго in-memory** (в оперативной памяти) без сохранения на диск.

## Запуск

```bash
npm install
npm start
```

Сервер запустится на `http://localhost:3000`

## Технологии

- Node.js + Express
- Multer (memory storage)
- JSZip + xml2js (обработка DOCX)
